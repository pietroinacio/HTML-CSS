Imagens

As imagens disponíveis na INTERNET normalmente tem direitos autorais, você pode pegar imagens sem direitos autorais em sites que servem como banco de imagens ou por meio da aba Ferramentas na pesquisa do Google, apertando nas opções que te dão o direito de usá-las.

Para adicionar uma imagem, basta usar a tag <img>, que não tem fechamento de tag e dois parâmetros automáticos, o src ou source e o alt ou texto alternativo.
Por exemplo:
img src="local da foto" alt="texto alternativo"

Há também uma forma de adicinar um favicon, que é o símbolo que aparece depois do nome da página na aba, para isso basta usar tag <link>, com a opção favicon. Ou seja usar a tag <link:favicon>

Formatações de textos

Algumas formatações de textos são os textos em negritos e em itálico. Há uma discussão sobre as tags obsoletas, que são as tags que existem desde a versão 4 do HTML, mas elas não estão corretas ao serem usadas, já que elas mudam a forma do texto e o HTML5 é focado somente na semântica.

Então por exemplo, para deixar o texto em negrito de forma não semântica em HTML, que não é recomendada pela W3C, se usa a tag <b>. Mas de forma correta e semântca, você pode usar a tag <strong>.

Já para deixar o texto em itálico, de forma semântica é preciso escrever a tag <em>, de forma não semântica e não recomendada pela W3C, o órgão oficial da linguagem HTML, a tag <i>.

Para deixar um texto marcado como um marca-texto, basta usar a tag <mark>.

Para marcar um texto excluído, basta usar a tag <del>, para marcar como inserido, basta usar a tag <ins>, mas também tem a tag <u> (não semântico) para sublinhado.

Para inserir coisas do tipo x2 e H2O, basta usar a tag <sup> e <sub> respectivamente.

CSS Inline

O CSS inline é praticamente uma forma de CSS dentro de tags HTML, é uma forma limitada e nada prático, mas é útil para mudanças momentâneas e pequenas.

CSS interno

É uma forma muito melhor que a forma inline, é praticamente uma forma de CSS dentro do seu arquivo HTML, dentro da área head do arquivo. Mas mesmo assim com linhas muito grandes, o arquivo HTML em si vai estar muito embaixo.

CSS externo

É a melhor forma de CSS, que é um arquivo separado somente focado no CSS, é acessado por meio do <link:css> dentro da área head de seu arquivo HTML.

Conclusão das CSS

Cada um tem suas vantagens e desvantagens, porém é sempre melhor evitar o estilo CSS Inline, pois deixa o arquivo HTML muito confuso e não é nada prático, usar o CSS interno somente se seu site tiver poucas páginas e quando não é preciso muito estilo. E finalmente, o mais recomendável é o CSS externo, pois serve para um site com muitas páginas e é muito mais versátil que as outras versões.
